# Nanodent – Dental Clinic

Website of dental clinic **Nanodent**, Plovdiv, Bulgaria. Built with [Jekyll](https://jekyllrb.com/) and deployed via GitHub Pages.

## Running Locally

**Requirement:** Ruby installed on your system

```bash
gem install bundler # if you don't have bundler
bundle config set --local path 'vendor/bundle'
bundle install
bundle exec jekyll serve
```

Open [http://localhost:4000](http://localhost:4000) (Or whatever URL appears in the terminal)
